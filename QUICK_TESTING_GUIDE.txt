================================================================================
                    KEY PATTERNS TO OBSERVE
================================================================================

1. ML & DL Models:
   - Perfect on FAKE news (100% accuracy)
   - Struggle with REAL news (20% accuracy)
   - Reason: Training data bias (2015-2017 political news)

2. LLM (LLaMA 3.3 70B):
   - Balanced performance
   - Better generalization
   - Provides detailed reasoning
   - More conservative confidence (60-90%)

3. Agent Swarm:
   - Cross-verification approach
   - Shows evidence from 20 sources
   - Slower but thorough
   - Good for borderline cases

4. Confidence Scores:
   - FAKE news: 90-100% (high confidence)
   - REAL news: 60-90% (varies by model)
   - INCONCLUSIVE: 40-60% (not enough evidence)

================================================================================
                    COMMON ISSUES & SOLUTIONS
================================================================================

Issue: Backend not responding
Solution: Check if python backend_api.py is running on port 5000

Issue: Frontend shows error
Solution: Check if npm run dev is running on port 3000

Issue: Agent Swarm takes too long
Solution: Normal! It searches 20 websites. Wait 20-40 seconds.

Issue: LLM shows error
Solution: Check .env file has GROQ_API_KEY

Issue: Models disagree on verdict
Solution: Normal! Different models have different strengths.
          - ML/DL: Pattern matching (fast, biased)
          - LLM: Semantic understanding (slow, balanced)
          - Swarm: Cross-verification (slowest, thorough)

Issue: All models say FAKE for REAL news
Solution: Training data bias. Models trained on 2015-2017 news.
          New topics (2024) not in training data.
          LLM and Agent Swarm should still detect correctly.
